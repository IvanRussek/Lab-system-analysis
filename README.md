## Lab-system-analysis

Мини-набор задач по “системному анализу” (граф/дерево отношений).

### Структура

- `task0/` — построение **матрицы смежности** по списку рёбер.
- `task1/` — построение матриц отношений **r1..r5** для дерева (родитель/потомок/братья).
- `task2/` — расчёт энтропийной оценки (возвращает \(H\) и нормированную \(h\)).

### Как запускать локально

Все задачи — обычные Python-скрипты, их можно запускать напрямую:

```bash
python task0/task.py
python task1/task.py
python task2/task.py
```

### Интерфейсы функций (для проверки)

#### `task0/task.py`

- `main(s: str) -> list[list[int]]` — принимает CSV-текст со списком рёбер (`"1,2\n1,3"`), возвращает матрицу смежности из `0/1`.
- `task(s: str) -> list[list[int]]` — алиас на `main`.

#### `task1/task.py`

- `main(s: str, e: str) -> (r1, r2, r3, r4, r5)` — возвращает 5 булевых матриц.

#### `task2/task.py`

- `main(s: str, e: str) -> (H, h)` — возвращает кортеж `(H, h)` округлённый до 1 знака.
- `task(s: str, e: str) -> (H, h)` — алиас на `main`.
